Maquarie University

Designing an intutive 'Getting Started' toolkit to help new Macquarie students seamlessly navigate their university life

Services

UX UI Design

Platform

Website

My Role

UX UI Designer

Duration

09/2022 - 11/2022

Challenge

There is a disconnect between the current "getting started" webpage on the student portal and the content in the EDMs (emails) that students receive, leading to confusion and frustration. Additionally, the existing "getting started" information lacks a clear, step-by-step guide, with content that doesnâ€™t follow a logical or sequential flow.

Discovery

Through foundational user research, codesign workshops with staff from multiple departments, we streamlined the user journey and various university service touch points, and therefore, and we analysed the potential gap between the current state ("as-is") and the desired future state ("to-be"). As a result, one of the main insights that were discovered is the existing Macquarie website lacks a step-by-step guide that would help to navigate incoming students to follow.

↑ COMPETITOR WORKSHOP

Development

Following the discovery phase, it was essential to develop a time-specific, sequential "Getting Started Checklist" that serves as an easy-to-follow, step-by-step guide to ennsure the content on the checklist page aligns with the student EDMs to prevent confusion.

Outcome

To address the disconnect between the "Getting Started" webpage and student EDMs, we created a time-specific, sequential "Getting Started Checklist" that provides a clear, step-by-step guide for new students. By aligning the content between the checklist page and EDMs, we ensured consistency across both channels, reducing confusion. This solution improved the clarity of information and created a more seamless onboarding experience for Macquarie University students.
